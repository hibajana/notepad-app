# Notes Web Application

A simple and modern web-based note taking application built with HTML, CSS, and JavaScript.

## Features
- Create, edit and delete notes
- Automatic saving with LocalStorage
- Search notes
- Filter by date
- Responsive design (mobile & desktop)

## Technologies
- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage API

## How to Use
1. Open `index.html` in your browser
2. Write a title and content
3. Save your note
4. Edit or delete anytime
